## 1. Link and Preview

Project link is available at [Contacts App](https://github.com/)

## 2. Description

The Contacts App is a web application that helps you to store all contacts in one place. You can add and delete a contact.

## 5. Must Haves

- A user should be able to see contacts' info on one page.
- A user should be able to add a contact and the contact must be added to the list.
- A user should be able to delete a contact.

## 6. Nice-to Have

- A user can see the about page and return homepage.
- If user does not enter a text, there must be an error.
